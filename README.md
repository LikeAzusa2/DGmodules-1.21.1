# DG Modules

Additional modules for **Draconic Evolution**.

DG Modules is an **unofficial addon mod** that adds extra experimental modules
and related mechanics to Draconic Evolution.

---

## 简介（中文）

DG Modules 是一个 **非官方的龙之进化（Draconic Evolution）附属模组**，  
用于添加额外的实验性模块及相关机制。

本项目主要面向学习、拓展与实验用途。

---

## Requirements / 依赖

- Minecraft **1.21.1**
- **NeoForge**
- **Draconic Evolution**
- **Brandon's Core**
- **Curios API**

---

## Usage Notes

- This mod is provided **as-is**.
- Compatibility with other mods is **not guaranteed**.
- Before adding this mod to an existing world, **please back up your save files**.

---

## 使用说明（中文）

- 本模组按 **现状（as-is）提供**
- **不保证** 与其他模组完全兼容
- 在加入已有存档前，**请务必备份你的世界存档**
